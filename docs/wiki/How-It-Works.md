# How It Works

Mandate turns open-ended wallet access into constrained execution authority.

```text
Create Mandate
  -> Define Limits
  -> Agent Executes
  -> Move Policy Checks
  -> DeepBook Execution
  -> On-chain Logs
```

## Lifecycle

Sign once -> Execute within limits -> Revoke/Expire -> Withdraw funds

## What The Owner Controls

- Budget
- Per-transaction cap
- Protocol scope
- Expiration window
- Revocation

Mandate keeps wallet ownership with the user while allowing the agent to operate inside a clear policy boundary.
